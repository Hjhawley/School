DOM - Document Object Model

Basic HTML tags:

<a> 		- define a hyperlink
<p> 		- define a paragraph
<nav> 	- navigation links
<head> 	- metadata / information for the document
<header> 	- top of the website
<h1>		- heading1 (<h2> through <h6>, etc)
<main> 	- main content area
<footer> 	- the bottom
<ul>		- define an unordered list
<ol>		- define an ordered list
<li>		- list item
<br>		- line break
<b>		- bold text
<strong>	- important text
<i>		- alternate voice or mood
<em>		- emphasized text

Languages we will focus on:
HTML - hypertext markup language; the layout, the bone structure
CSS - cascading style sheets; the clothing

Others:
XML - allows user to define their own tags
XHTML - combines HTML and XML
HTML ver.5 combined basically everything
